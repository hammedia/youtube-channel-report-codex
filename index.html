YouTube Studio CSV 받는 방법

💡 YouTube Studio에서 내보내기 시 ZIP 파일로 다운로드하면 채널명과 분석 기간을 자동으로 인식합니다. ZIP 업로드를 권장합니다.

1. YouTube Studio에서 분석하려는 채널의 분석 메뉴로 들어갑니다.

2. 기간을 선택합니다. 처음 비교용으로는 보통 최근 90일을 추천합니다.

3. 고급 모드로 들어간 뒤 오른쪽 위 내보내기에서 두 옵션 중 CSV를 선택합니다.

4. 표 데이터.csv는 필수이고, 총계.csv와 차트 데이터.csv를 함께 올리면 추가 차트가 열립니다. 업로드 순서는 상관없습니다.

표 데이터 CSV를 올리면, 위에서 바로 요약 보고서를 만들고 아래에 차트와 영상별 해설을 보여줍니다.
