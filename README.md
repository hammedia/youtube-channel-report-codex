# 유튜브 채널 분석 리포트 툴

YouTube Studio CSV를 넣으면 조회수, CTR, 시청 지속률,
구독 전환 흐름을 한 번에 보고서 형태로 정리해주는 분석 툴입니다.
숫자만 보여주는 게 아니라 지금 채널에서 가장 먼저 고쳐야 할 것까지
읽히는 순서로 정리합니다.

🔗 https://hammedia.github.io/youtube-channel-report-codex/

---

## 특징

- CSV는 서버에 저장되지 않고 브라우저 안에서만 분석됩니다
- 별도 설치 없이 링크 하나로 바로 사용 가능합니다
- 분석 결과를 독립 HTML 파일로 저장해 클라이언트에게 전달할 수 있습니다

---

## CSV 준비 방법

1. YouTube Studio 접속 → 분석 메뉴
2. 기간 선택 (처음 사용 시 최근 90일 추천)
3. 고급 모드 → 오른쪽 상단 내보내기 → CSV 선택
4. 아래 파일을 저장합니다

| 파일명 | 필수 여부 |
|--------|----------|
| 표 데이터.csv | 필수 |
| 총계.csv | 선택 (일별 조회수 차트) |
| 차트 데이터.csv | 선택 (상위 영상 흐름 차트) |

---

## 사용 방법

1. 위 링크 접속
2. CSV 파일을 드래그하거나 클릭해서 업로드
3. 분석 결과가 자동으로 표시됩니다

---

## 클라이언트에게 리포트 전달하는 방법

분석 결과가 표시된 상태에서 상단의 **리포트 저장** 버튼을 클릭합니다.

클릭하면 순서대로 세 가지를 물어봅니다.

1. **채널명** — 입력하면 리포트 제목에 반영됩니다. 건너뛰려면 확인만 누르세요.
2. **받는 분 이름** — 입력하면 리포트 상단에 수신자로 표시됩니다. 건너뛰려면 확인만 누르세요.
3. **포함할 영상 수** — 기본값은 10개입니다. 더 많이 담으려면 숫자를 입력하세요.

입력이 끝나면 `[채널명]-youtube-report-YYYYMMDD.html` 파일이 자동으로 다운로드됩니다.

이 파일을 카카오톡, 이메일, 슬랙 등으로 전달하면
클라이언트가 브라우저에서 바로 열어볼 수 있습니다.
별도 프로그램 설치는 필요하지 않습니다.

## 분석 항목

- 채널 컨디션 점수 (CTR · 지속률 · 구독 전환 종합)
- 핵심 지표 요약
- 쇼츠 vs 일반 영상 포맷 비교
- 일별 조회수 흐름
- 조회수 상위 10개 영상
- 영상 길이 구간별 평균 성과
- 상위 영상 날짜별 흐름
- 잘된 영상 Top 3 / 아쉬운 영상 Bottom 3
- 영상별 CTR · 지속률 · 구독 전환 개별 해설
- AI 활용 프롬프트 (CTR / 지속률 / 구독 전환 개선용)
