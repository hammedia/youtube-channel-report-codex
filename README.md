# 유튜브 채널 분석 리포트 툴

YouTube Studio에서 받은 ZIP 파일 또는 CSV를 넣으면
조회수·CTR·시청 지속률·구독 전환 흐름을 한 번에 분석합니다.

숫자만 보여주는 게 아니라 지금 채널에서 가장 먼저 고쳐야 할 것까지
읽히는 순서로 정리합니다.

채널을 직접 운영하는 유튜버와
클라이언트 채널을 관리하는 대행사·프리랜서 모두 사용할 수 있습니다.

- 유튜버라면: 내 채널 데이터를 넣고 무엇부터 고쳐야 할지 바로 확인
- 대행·관리라면: 분석 후 클라이언트 이름 넣어서 리포트 파일로 바로 전달

🔗 https://hammedia.github.io/youtube-channel-report-codex/

---

## 특징

- ZIP 파일 업로드 시 채널명·분석 기간 자동 인식
- CSV는 서버에 저장되지 않고 브라우저 안에서만 분석됩니다
- 별도 설치 없이 링크 하나로 바로 사용 가능합니다
- 분석 결과를 클라이언트용 리포트 HTML로 저장해 바로 전달할 수 있습니다

---

## 파일 준비 방법

### ✅ 권장: ZIP 파일로 업로드

1. YouTube Studio 접속 → 분석 메뉴
2. 기간 선택 (처음 사용 시 최근 90일 추천)
3. 고급 모드 → 오른쪽 상단 내보내기 → **CSV 다운로드 (ZIP)**
4. 다운로드된 ZIP 파일을 그대로 업로드

ZIP 파일명에 채널명과 분석 기간이 포함되어 있어
자동으로 인식합니다. 별도 입력이 필요 없습니다.

> ⚠️ 폴더 업로드는 브라우저 보안 정책상 지원되지 않습니다.

### CSV 파일 개별 업로드

ZIP 없이 CSV 파일을 직접 업로드할 수도 있습니다.

| 파일명 | 필수 여부 |
|--------|----------|
| 표 데이터.csv | 필수 |
| 총계.csv | 선택 (일별 조회수 차트) |
| 차트 데이터.csv | 선택 (상위 영상 흐름 차트) |

이 방식은 리포트 저장 시 채널명을 직접 입력해야 합니다.

## 사용 방법

1. 위 링크 접속
2. ZIP 파일 또는 CSV 파일을 드래그하거나 클릭해서 업로드
3. 분석 결과가 자동으로 표시됩니다

---

## 클라이언트에게 리포트 전달하는 방법

분석 결과가 표시된 상태에서 상단의 **리포트 저장** 버튼을 클릭합니다.

클릭하면 순서대로 네 가지를 물어봅니다.

1. **채널명** — ZIP 업로드 시 자동 입력됩니다. CSV 업로드 시 직접 입력하세요.
2. **받는 분 이름** — 입력하면 리포트 상단에 수신자로 표시됩니다. 건너뛰려면 확인만 누르세요.
3. **보내는 분 이름** — 입력하면 리포트 상단에 발신자로 표시됩니다. 건너뛰려면 확인만 누르세요.
4. **포함할 영상 수** — 기본값은 10개입니다. 더 많이 담으려면 숫자를 입력하세요.

입력이 끝나면 `[채널명]-youtube-report-YYYYMMDD.html` 파일이
자동으로 다운로드됩니다.

이 파일을 카카오톡, 이메일, 슬랙 등으로 전달하면
클라이언트가 브라우저에서 바로 열어볼 수 있습니다.
별도 프로그램 설치는 필요하지 않습니다.

## 분석 항목

### 라이브 툴에서 확인 가능
- 채널 컨디션 점수 (CTR · 지속률 · 구독 전환 종합)
- 핵심 지표 요약
- 쇼츠 vs 일반 영상 포맷 비교
- 일별 조회수 흐름
- 조회수 상위 10개 영상
- 영상 길이 구간별 평균 성과
- 상위 영상 날짜별 흐름
- 잘된 영상 Top 3 / 아쉬운 영상 Bottom 3
- 영상별 CTR · 지속률 · 구독 전환 개별 해설
- AI 활용 프롬프트 (CTR / 지속률 / 구독 전환 개선용)

### 저장 리포트에 포함되는 항목
AI 활용 프롬프트를 제외한 모든 분석 항목.
클라이언트에게 전달할 때는 영상 수를 선택할 수 있습니다 (기본 10개).
