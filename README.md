# YouTube Channel Report

브라우저에서 YouTube Studio CSV를 업로드해 채널 성과를 보고서 형태로 확인하는 정적 HTML 도구입니다.

분석 결과를 숫자로만 보여주는 것이 아니라, 사용자가 바로 실행할 수 있도록 우선 개선 액션과 AI 프롬프트를 함께 제공합니다.

## 주요 기능

- 총 조회수, CTR, 시청 지속률, 구독 전환율 요약
- 지금 가장 먼저 해야 할 개선 액션 3가지를 우선순위 순으로 자동 정리
- 일별 조회수, 상위 영상, 카테고리별 성과 차트
- 영상별 해설 목록과 10개, 20개, 30개 단위 페이지 보기
- CTR, 시청 지속률, 구독 전환 개선용 AI 프롬프트 3개
- 프롬프트 복사하기 버튼

## CSV 받는 방법

1. YouTube Studio에서 분석하려는 채널의 분석 메뉴로 들어갑니다.
2. 기간을 선택합니다. 처음 비교용으로는 보통 최근 90일을 추천합니다.
3. 고급 모드로 들어간 뒤 오른쪽 위 내보내기에서 CSV를 선택합니다.
4. `표 데이터.csv`는 필수이고, `총계.csv`와 `차트 데이터.csv`를 함께 올리면 추가 차트가 표시됩니다.

## 사용 방법

1. 웹페이지를 엽니다.
2. CSV 파일을 드래그하거나 클릭해서 업로드합니다.
3. 파일 순서는 상관없습니다. 페이지가 CSV 구조를 보고 자동으로 구분합니다.
4. Executive Summary와 `지금 가장 먼저 해야 할 것`을 먼저 확인합니다.
5. 필요한 AI 프롬프트를 복사해 ChatGPT, Claude, Gemini 등에 붙여넣어 실행합니다.

## 개인정보 안내

CSV는 서버에 저장하지 않고 사용자의 브라우저 안에서만 읽습니다.

단, 이 웹페이지 자체는 GitHub Pages에 공개되어 있으며 Chart.js와 PapaParse를 CDN에서 불러옵니다.
